01 — Platform Vision

A performance-driven adaptive learning loop

Cautie Learn Hub turns every study action into measured signal. Tools feed performance data into an adaptive engine that continuously rewrites each learner's plan — closing the gap between studying and actually being ready.

What makes it world-class

One dataset

Every tool, one mastery score

A flashcard miss and a quiz miss on the Krebs Cycle update the same topic node. No siloed progress.

Always current

The plan rewrites itself

Finish a session at 78% and tomorrow's plan already reflects it — no manual scheduling.

Outcome-first

Readiness, not streaks

We project a score for Mar 28 · AP Biology, then drive every recommendation toward it.

Feature specs
  • Unified mastery graph — all five tools write to one per-topic mastery model (0–100).
  • Readiness projection — Bayesian estimate of exam-day score with confidence band.
  • Continuous adaptation — plan recomputed after every completed tool run.
  • Signal richness — captures accuracy, latency, self-rated confidence, and recency decay.
  • Zero-config start — paste material, pick an exam date, the engine builds the first plan.
02 — Information Architecture

How everything is organized

A clean four-level hierarchy keeps learners oriented: a Subject contains Studysets, each Studyset is studied across Sessions, and each Session is made of individual Tool Runs that emit performance data.
