Flashcard types & the new options panel

Section 1 shows the proposed Flashcards options screen, rebuilt with the exact same layout language as the Quiz options screen (breadcrumb header, left type list, right settings rail). The new piece is the field-pair swap control — two labels with a swap button between them, so the same "two-sided card" mechanic can represent Term↔Definition, Event↔Date, Question↔Answer, etc. Whatever pair you pick is just sent as two field labels in the AI generation request — the AI fills them in per card, no extra logic needed per type.

Section 2 is a gallery of every flashcard type I could find used in real tools (Anki, Quizlet, RemNote, Notion flashcards) plus the ones already proposed earlier in TOOL_SETTINGS_MOCKUP.md. Some are just label-pair presets (swap-style). Others — Cloze, Image Occlusion, Multiple Choice, True/False, Matching — need a genuinely different card shape, not just a label swap, so they're shown separately with their own little demo.
