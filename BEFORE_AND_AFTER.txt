CalDAV Integration: Before & After Comparison
═════════════════════════════════════════════════════════════════════════════

BEFORE: Manual ICS Import/Export System
───────────────────────────────────────────────────────────────────────────────

Agenda Page UI:
  [View] [Classes] [Settings] [Export] [Import] [+ Add Activity]

User Flow:
  1. Click "Export" → Downloads ICS file to computer
  2. Open Apple Calendar/Google Calendar manually
  3. Import ICS file through their calendar app
  4. Close Cautie, go to calendar app
  5. NO sync when Cautie events change
  6. To update: Repeat entire process
  7. Click "Import" → Select file from computer
  8. Parses events and adds to agenda
  9. One-time import only

Features:
  ✗ Manual file management
  ✗ No real-time sync
  ✗ One-way data flow
  ✗ Requires calendar app switch
  ✗ No event linking
  ✗ File-based storage

Setup Requirements:
  ✗ Manual OAuth configuration
  ✗ API key management
  ✗ Google Cloud Console setup
  ✗ ICS format understanding
  ✗ Manual syncing discipline


AFTER: Modern CalDAV Real-Time Sync
───────────────────────────────────────────────────────────────────────────────

Agenda Page UI:
  [View] [Classes] [Settings] [Connect Calendar] [+ Add Activity]

User Flow:
  1. Click "Connect Calendar" button
  2. Beautiful dialog opens
  3. Select calendar provider (Apple/Google/Outlook/CalDAV)
  4. Enter email/password
  5. Click "Connect"
  6. Account connected and syncing automatically
  7. Events appear in Cautie automatically
  8. Any changes sync both ways
  9. No file management needed

Features:
  ✓ Automatic real-time sync
  ✓ Bi-directional data flow
  ✓ No app switching needed
  ✓ Event linking preserved
  ✓ Scheduled automatic syncs
  ✓ User credentials only (no API keys)

Setup Requirements:
  ✓ Just click button, enter password
  ✓ No API keys needed
  ✓ No OAuth setup required
  ✓ No developer console access
  ✓ Automatic sync (no user action)

SETUP COMPLEXITY
───────────────────────────────────────────────────────────────────────────────

Before (ICS):
  1. Go to Google Cloud Console
  2. Create OAuth 2.0 credentials
  3. Copy API keys
  4. Add to environment variables
  5. Configure redirect URLs
  6. Manage API quotas
  → Total: 30+ minutes, technical knowledge required

After (CalDAV):
  1. Click "Connect Calendar"
  2. Enter your calendar email
  3. Enter your app-specific password
  4. Click "Connect"
  → Total: 30 seconds, no technical knowledge needed
