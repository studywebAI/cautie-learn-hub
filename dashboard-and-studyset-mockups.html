Design Notes & Layout Patterns

Hero + Quick Stats
Welcome banner at top with key metrics in 3-column grid. Good for overview dashboards where users need immediate context.
Timeline View
Chronological list of events/deadlines with priority indicators. Ideal for showing upcoming work and due dates.
Sidebar Navigation
Left sidebar for course/class selection with contextual stats on right. Helps manage multiple classes or subjects.
Priority Stack
Colored left border indicating urgency. Stacked cards show task priority visually without overwhelming interface.
Analytics Grid
Multi-column stat layout with charts. Good for performance data, trends, and comparative metrics across classes.
Grading Queue
Left-aligned task list with status indicators. Prioritizes pending work for teachers to action items first.
Multi-Class Panel
Stacked class cards with key metrics. Gives teachers overview of all their classes in one view.
StudySet Workflow
Step indicator at top, form fields, material list. 4-step process: Name → Upload → Customize → Review.
Color Coding
Red/danger for urgent • Orange/warning for soon • Green/success for complete • Blue/info for neutral. Status badges on all items.
Theme Match
All layouts use site colors: Olive green (#7f8962) for primary, light grays for surfaces, proper typography hierarchy.
