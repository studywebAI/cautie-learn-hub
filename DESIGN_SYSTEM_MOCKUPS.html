Section 06
Proposed Fixes — Exact Code Changes
Exactly what needs to change in globals.css and tailwind.config.ts to fix all the inconsistencies. These are the minimum changes for maximum impact.
Fix 1 — Font: Change @import to Inter. Update --font-ui to "Inter". Update tailwind.config.ts font-sans to ["Inter", ...fontFamily.sans]. One font, no fallback chaos.
Fix 2 — Font weights: Remove * { font-weight: 400 !important }. Add proper rules: body = 400, labels = 500, headings = 600. Fix b, strong to 600. This alone will make the site look 10× better.
Fix 3 — Type scale: Replace 4 variables with 5: --type-micro: 12px, --type-small: 13px, --type-body: 14px, --type-section: 17px, --type-page: 22px. Fix text-xl to 20px (not same as text-lg).
Fix 4 — Border radius: Set --radius: 0.5rem (8px). This makes rounded-lg = 8px, rounded-md = 6px. Use rounded-lg for inputs/buttons, rounded-xl for cards. Consistent everywhere.
Fix 5 — Colour tokens: Find+replace all bg-[#7f8962], text-[#7f8962], bg-[#6f7851] with bg-[var(--accent-brand)] / text-[var(--accent-brand)] / bg-[var(--accent-brand)]/90.
Fix 6 — font-500/font-600: Find+replace all font-500 → font-medium and font-600 → font-semibold in class-settings-redesigned.tsx (and anywhere else used).
