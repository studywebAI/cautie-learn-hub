Flashcard layouts

Browse by family → pick a layout to view it full-size

Every layout below shows the same sample card ("Mitochondria") so only the shape/interaction changes. Click a name on the left — the matching layout opens full-size on the right with a plain-language note on what it's for.

1 · Regular flashcards

The classic front ↔ back card. These 4 differ only in HOW the back is revealed.

2 · Fill-in-the-blank (Cloze)

The term is hidden INSIDE a full sentence instead of standing alone. These 4 differ in how you fill the gap.

3 · Context-aware cards

The original source sentence travels with the card, not just the isolated term. These 4 differ in how prominent that source text is.

4 · Mnemonic (memory-trick) cards

Bundles a memory trick (rhyme / association / image-in-words) with the card. These 4 differ in how prominent that trick is.

5 · "Type the answer" study mode

Not a separate card style — a STUDY MODE: you see one side, type your own recall attempt, then flip to compare. These 4 differ in what you're asked to recall and how you're checked.
