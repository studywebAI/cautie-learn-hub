Interactive Tools: Layout & Interaction Designs

Setup: These tools provide visual/interactive ways for students to organize, map, and timeline information. Designs below show layout approach, toolbar placement, and interaction model.
🧠 Mindmap Tool
Interactive mind mapping with central node, radial child arrangement, drag-to-add, click-to-edit. Features: auto-position with no overlaps (force-directed layout), export as JSON, color coding by depth, smooth pan/zoom.
Toolbar (Top)
+ Add ✏️ Edit 🗑️ Delete 💾 Download ⚙️ Settings
Canvas (Main Area)
SVG Canvas: Central node in middle, children positioned radially, connected by lines
Interaction Flow:
  1. Create: Click canvas → "Create new mindmap" form appears (title input)
  2. Add child: Select node → "+ Add" button → text input → saves automatically
  3. Edit: Click node → inline edit mode or modal (for longer text)
  4. Delete: Select node → "Delete" button → confirm
  5. Pan/Zoom: Mouse wheel to zoom, drag canvas to pan
  6. Export: Download button → JSON file with full structure
  7. Settings: Toggle layout tightness, color scheme, labels on/off
Technical Details:
  • Layout: Force-directed (d3-force) or hierarchical radial positioning to prevent node overlap
  • Rendering: SVG with React for reactive updates
  • Depth colors: Each level gets different color from palette
  • No overlaps: Algorithm calculates optimal radius per depth to space nodes evenly
  • Smooth transitions: CSS transitions on node position changes
⏱️ Timeline Tool
Linear timeline with events positioned by date, draggable to reorder, color-coded. Features: auto-sort by date, inline edit, add/delete events, smooth animations, emoji color codes, drag-to-reorder.
Controls (Top)
+ Add Event 📥 Import 💾 Export
Timeline Display
Event title
Date & time
Description...
Next event
Date & time
Interaction Flow:
  1. Add event: Click "+ Add Event" → form appears (title, date/time, description)
  2. Edit event: Double-click event card → inline edit form appears
  3. Delete: Hover → trash icon → confirm delete
  4. Reorder: Drag event by handle (grip icon) → visual feedback
  5. Color-code: Each event gets auto-assigned color from palette
  6. Auto-sort: Events always sort by date (earliest first)
  7. Export: Download as JSON with event metadata
Technical Details:
  • Sorting: Automatic date-based ordering, user can drag to reorder within view
  • Rendering: Vertical timeline with center line, events alternate left/right (or all on right)
  • Drag behavior: React DnD or native drag API, smooth visual feedback
  • Animations: Fade in/out on add/delete, slide transitions on reorder
  • Color dots: Visual markers on timeline line, matching event color
  • Dates: ISO format storage, localized display (en-GB by default)

Design Selection Guide: Choose one design per tab (or multiple if you want to implement variants).

Example implementations: GRP-4, CHT-2, ATT-3, GRD-T2, GRD-S3, SCH-1 or any combination.

Tools: Mindmap = SVG canvas with radial node layout. Timeline = Vertical timeline with drag-to-reorder.

All designs use consistent color palette: #7f8962 (sage primary), #d8956c (tan), #6b9fbf (ocean), etc.
