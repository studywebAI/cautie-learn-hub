# 🎓 CORRECT Tool Settings & Modes (Cautie Philosophy)

**Based on: Set-specific learning, mastery-based progression, continuous flow**

---

## 📚 FLASHCARDS TOOL

### PHILOSOPHY
- Study set is **CLOSED** (no spaced repetition intervals)
- Learn until **100% mastery** (all cards correct)
- Study session = study until done OR quit
- No cards come back unless source text changes
- Progress shown: Cards remaining / Total

## 🧠 QUIZ TOOL

### PHILOSOPHY
- **Continuous flow** of questions (not fixed number)
- Questions get **harder as you master** easier ones
- System adapts difficulty in real-time
- **Keep going until mastery** = 3+ correct in a row at hardest level
- New questions always from **same topic** (not random)
- "How many questions?" = irrelevant (you decide when to stop)
