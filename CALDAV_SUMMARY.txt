Supported Providers:
  🍎 Apple iCloud Calendar (caldav.icloud.com)
  🔵 Google Calendar (caldav.google.com)
  📧 Microsoft Outlook (outlook.office365.com)
  🔧 Custom CalDAV Servers (user-provided URL)

No API Keys Needed:
  ✅ Uses CalDAV protocol (standard HTTP basic auth)
  ✅ Works with user's own credentials
  ✅ No OAuth app registration required
  ✅ No API key management needed

User Security:
  ✅ Passwords stored per-user
  ✅ Row-level security prevents cross-user access
  ✅ Credentials never exposed in API responses
  ⚠️  Use app-specific passwords (not regular passwords)
